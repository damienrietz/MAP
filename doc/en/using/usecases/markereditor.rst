Marker Editor
=============

This section illustrates how you can use the Marker Editor Plugin. 

MapStore allows to dynamically add markers to the map. These marker can be injected using MapStore in an iframe, or edited/viewed using the Marker Editor plugin.

Using the Marker Editor Plugin
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

The Marker Editor Plugin allows creating, editing and viewing markers on the map. The marker can have an icon, a label and some content to show when you click on it. You can save and restore these markers as simple text.

Open the `MapComposer <http://localhost:8081/mapcomposer/viewer?locale=en&config=markerEditor>`__.

	.. figure:: img/marker_editor_home.png
	
				MapComposer in Viewer template with Marker Editor Plugin.

 Close the help window and you will see the map with a panel on the right.
 
	.. figure:: img/marker_editor_panel.png
	
				The main Marker Editor panel.

This Panel allows to:
 * Create new markers
 * Modify/Delete existing markers
 * import and export marker configurations

the module on the top allows to create, edit and save marker settings. The list on the bottom shows already created markers.

Create a New Marker
^^^^^^^^^^^^^^^^^^^
#. Activate the button with a mouse pointer as icon |coordinatebutton|. This control allows to get the coordinates of the next click on the map. So click on the map and you will see the coordinates of the marker in the two text fields around the button and a red cross on the map. This cross indicates the position of the marker.

    .. |coordinatebutton| image:: img/marker_editor_coordinate_button.png
        :height: 20
        :width: 20

    .. figure:: img/marker_editor_coordinates.png
	
				Add the coordinates of the Marker to the map.
                
#. Choose the marker editor clicking on the button with the red marker |markerbutton|. Select the blue marker and click on **Use this Marker**
       .. figure:: img/marker_editor_window.png
	
				Choose the marker from the window.
                
    .. |markerbutton| image:: img/marker_editor_marker_button.png
        :height: 20
        :width: 20

                
   .. note:: The marker listed in the window above are configured in a separated file that contains the picture paths and sizes. You can find the default configuration at: ``app\static\data\markerconfigurations.js``
    
#. Set Label field with 1, Title field with "My Marker", and Content field with "Sample Text".

    .. figure:: img/marker_editor_complete_configuration.png
	
				A complete marker configuration.
                
#. Click on "Add To The Map". Your marker will be added on the map and it wil be also added to the table at the bottom of the panel.

   .. figure:: img/marker_editor_first_marker.png
	
				The Marker on the map and in the table.
                
If you click on the marker you can see a popup with the title and the content you edited.
   .. figure:: img/marker_editor_popup.png
	
				The popup with the marker.

To edit marker positon or settings select it on the list at the bottom of the panel, do your changes and click update. To delete it click on |removebutton| near the marker in the list.
    .. |removebutton| image:: img/marker_editor_marker_button.png
        :height: 20
        :width: 20
        
  .. figure:: img/marker_editor_modify.png
	
				Do your changes to the marker settings.

                
Export and import Marker
^^^^^^^^^^^^^^^^^^^^^^^^
The Markers created with thw Marker Editor can be saved/restored simply copying and pasting some text.

You can export your markers clicking on |exportbutton| and copying the text in the window.
   .. |exportbutton| image:: img/marker_editor_export_button.png
        :height: 20
		
  .. figure:: img/marker_editor_export_win.png
	
				Copy the text to save your markers.
                
You can import your markers again clickin on |importbutton|, pasting your code in the window and clicking on |importgeojson|
   .. |importbutton| image:: img/marker_editor_import_button.png
        :height: 20
   .. |importgeojson| image:: img/marker_editor_import_geojson_button.png
        :height: 20
        
  .. figure:: img/marker_editor_import_win.png
	
				Paste the text to save your markers.
